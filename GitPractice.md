# Введение в контроль версий

## Git. Работа с терминалом

###### Заметка: чтобы сделать commit, нужно обязательно сохраниться (ctrl + s), затем добавить файл в отслеживаемое (git add .\fileName)
###### Заметка: она не дает возможности переключаться между ветками или выполнять слияние разветвленной истории. Именно поэтому команда git branch тесно связана с командами git checkout и git merge
#### Кстати! Чтобы из репозитория снова сделать папку, достаточно всего лишь удалить скрытую папку «.git»
