# Руководство по работе с Git.

## Команды для работы с git

1. git init - инициализация репозитория
2. git add .\fileName - добавление файла к отсллеживанию
3. git commit -m "comment" - фиксация изменений непосредственно в репозитории
4. git status - статус текущего репозитория
5. git log - список коммитов репозитория
6. git checkout code number of commit - для перемещения по коммитам (версиям проекта)
